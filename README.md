## Työn aihe

Tietokantasovelluksen harjoitustyön aihe on kenkälista kenkien tietojen (merkki, malli, kenkätyyppi, kuvaus) tallentamiseen. Jokainen kenkä kuuluu yhteen tai useampaan kenkätyyppiin.
